Reflection

Quiz 1 acted as an early check on basic Java terms and rules. At first, JVM, JRE, and JDK appeared as names to recall — not ideas to grasp. But writing real programs, then fixing errors, made the differences useful.

Consider the JVM. It powers Java's "write once, run anywhere" feature. That one fact shows why the compile step exists at all.

Memory concepts proved hardest. But three mental images helped.
The Stack holds temporary data. A method ends? Stack clears it.
The Heap keeps objects longer. They stay until no longer needed.
The Method Area stores blueprints — class definitions.
Seeing these as separate containers made variable placement logical.

Operators taught a small but critical lesson. Prefix versus postfix increment? Swap them and output changes completely.

Short-circuit evaluation matters outside quizzes too. With &&, if the left side is false, the right side never runs. That behavior prevents errors like NullPointerException — a tool, not just a test point.

A common beginner error the quiz reinforced: assignment uses one equals sign (=); comparison uses two (==). Seeing both on the same test locked the difference permanently. Without this quiz's groundwork, later OOP topics would have been far harder to build on.

How it works

This assignment explores how variables store data differently depending on type. Primitive variables (like salary) hold actual values in stack memory, whereas reference variables (e1, e2, e3) store pointers to objects on the heap. When creating new Employee objects, memory gets allocated; reassigning references orphans old objects for garbage collection. Modifying an object's properties through one reference affects all references pointing to that same instance, but reassigning a reference doesn't impact others—it simply redirects that variable to a different heap location.

import java.util.Scanner;

public class BasicATMSystemHilario {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        double balance = 5000.0;
        int choice;

        do {
            // Display the Menu
            System.out.println("\n===== ATM MENU =====");
            System.out.println("1. Check Balance");
            System.out.println("2. Deposit");
            System.out.println("3. Withdraw");
            System.out.println("4. Exit");
            System.out.print("Select an option: ");
            
            choice = scanner.nextInt();

            // Process the selected option using a switch statement
            switch (choice) {
                case 1:
                    System.out.println("Your current balance is: " + balance + " pesos");
                    break;
                    
                case 2:
                    System.out.print("Enter amount to deposit: ");
                    double deposit = scanner.nextDouble();
                    if (deposit > 0) {
                        balance += deposit;
                        System.out.println("Successfully deposited " + deposit + " pesos.");
                    } else {
                        System.out.println("Invalid deposit amount.");
                    }
                    break;
                    
                case 3:
                    System.out.print("Enter amount to withdraw: ");
                    double withdrawal = scanner.nextDouble();
                    
                    // Validate withdrawal using if and else statements
                    if (withdrawal > 2000) {
                        System.out.println("Transaction failed: Maximum withdrawal per transaction is 2,000 pesos.");
                    } else if (withdrawal > balance) {
                        System.out.println("Transaction failed: Insufficient balance.");
                    } else if (withdrawal <= 0) {
                        System.out.println("Transaction failed: Invalid amount.");
                    } else {
                        balance -= withdrawal;
                        System.out.println("Successfully withdrew " + withdrawal + " pesos.");
                    }
                    break;
                    
                case 4:
                    System.out.println("Thank you for using the ATM. Goodbye!");
                    break;
                    
                default:
                    System.out.println("Invalid choice. Please select a valid option from the menu.");
            }
            
        } while (choice != 4); // Loop until Exit is selected
        
        scanner.close();
    }
}

import java.util.Scanner;

public class EnrollmentAssessmentSystemHilario {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter Student Name: ");
        String studentName = input.nextLine();

        String strand = "";
        boolean validStrand = false;
        
        while (!validStrand) {
            System.out.print("Enter Senior High School Strand (STEM, ABM, HUMSS, TVL, Others): ");
            strand = input.nextLine().trim(); 
            
            if (strand.equalsIgnoreCase("STEM") || 
                strand.equalsIgnoreCase("ABM") || 
                strand.equalsIgnoreCase("HUMSS") || 
                strand.equalsIgnoreCase("TVL") || 
                strand.equalsIgnoreCase("Others")) {
                validStrand = true; 
            } else {
                System.out.println("Invalid input! Please type exactly one of the options listed.");
            }
        }

        System.out.print("Enter Entrance Exam Score (0-100): ");
        double entranceExam = input.nextDouble();

        System.out.print("Enter Interview Score (0-100): ");
        double interviewScore = input.nextDouble();
        
        input.nextLine(); 

        System.out.print("Enter First Screening Result (Pass/Fail): ");
        String screeningResult = input.nextLine();

        double finalScore = (entranceExam * 0.60) + (interviewScore * 0.40);

        String admissionStatus;
        if (screeningResult.equalsIgnoreCase("Pass")) {
            if (finalScore >= 85) {
                admissionStatus = "Admitted with Scholarship";
            } else if (finalScore >= 75) {
                admissionStatus = "Admitted";
            } else if (finalScore >= 65) {
                admissionStatus = "Waitlisted";
            } else {
                admissionStatus = "Not Qualified";
            }
        } else {
            admissionStatus = "Not Qualified"; 
        }

        String recommendedProgram;
        switch (strand.toUpperCase()) {
            case "STEM": recommendedProgram = "BSIT / BSCS"; break;
            case "ABM": recommendedProgram = "BSBA"; break;
            case "HUMSS": recommendedProgram = "BSED / AB Communication"; break;
            case "TVL": recommendedProgram = "BSIT"; break;
            default: recommendedProgram = "General Program"; break;
        }

        System.out.println("\n===== ENROLLMENT ASSESSMENT RESULT =====");
        System.out.println("\nStudent Name: " + studentName);
        System.out.println("SHS Strand: " + (strand.equalsIgnoreCase("Others") ? "Others" : strand.toUpperCase())); 
        System.out.printf("Final Evaluation Score: %.2f\n", finalScore);
        System.out.println("Admission Result: " + admissionStatus);
        System.out.println("Recommended Program: " + recommendedProgram);

        input.close();
    }
}

import java.util.Scanner;

public class ExpenseTrackerHilario {

    // Method 1: void – display program title 
    static void displayTitle() {
        System.out.println("=========================================");
        System.out.println("      PERSONAL EXPENSE TRACKER          ");
        System.out.println("=========================================");
    }

    // Method 2: non-void – compute total of all expense categories 
    static double computeTotal(double food, double transportation, double other) {
        return food + transportation + other;
    }

    // Method 3: non-void – check if total exceeds the budget 
    static String checkBudget(double total, double budget) {
        if (total > budget) {
            return "OVER BUDGET x  (exceeded by PHP " +
                   String.format("%.2f", total - budget) + ")";
        } else if (total == budget) {
            return "EXACT BUDGET -- You spent exactly your budget!";
        } else {
            return "WITHIN BUDGET";
        }
    }

    // Method 4: void – display the full summary 
    static void displaySummary(String name, double food, double transportation,
                               double other, double budget, double total, String status) {

        double remaining = budget - total;

        System.out.println("\n----- EXPENSE SUMMARY FOR " + name.toUpperCase() + " -----");
        System.out.printf("  Food              : PHP %,.2f%n", food);
        System.out.printf("  Transportation    : PHP %,.2f%n", transportation);
        System.out.printf("  Other Expenses    : PHP %,.2f%n", other);
        System.out.println("-----------------------------------------");
        System.out.printf("  Total Expenses    : PHP %,.2f%n", total);
        System.out.printf("  Budget Limit      : PHP %,.2f%n", budget);
        System.out.println("  Budget Status     : " + status);

        if (remaining > 0) {
            System.out.printf("  Remaining Budget  : PHP %,.2f%n", remaining);
            System.out.println("\n  Great job, " + name + "! You stayed within budget.");
        } else if (remaining == 0) {
            System.out.println("  Remaining Budget  : PHP 0.00");
            System.out.println("\n  Wow, " + name + "! You spent exactly your budget!");
        } else {
            System.out.printf("  Over by           : PHP %,.2f%n", Math.abs(remaining));
            System.out.println("\n  Hey " + name + ", try to cut back next time!");
        }
        System.out.println("=========================================");
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        displayTitle();
        // ... input and method calls ...
        sc.close();
    }
}
